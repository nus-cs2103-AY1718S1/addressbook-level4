* This is a desktop Sales Navigator application. It has a GUI but most of the
user interactions happen using a CLI (Command Line Interface).
* It is a Java application intended for sales executives to manage their clients and events
* What's new in v1.2:
** A fresh look with the updated UI
** Added features such as editing and finding events
** Added enhancements such as sorting, saving company, work contact and position information for contacts
** More test cases, including automated GUI testing.
** Support for _Build Automation_ using Gradle and for _Continuous Integration_ using Travis CI.
