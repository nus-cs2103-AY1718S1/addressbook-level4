* This is a desktop Tourist Book application. It has a GUI but most of the user interactions happen using a CLI (Command Line Interface).
* It is a Java application intended for tourists coming to Singapore for sightseeing.
* It provides a comprehensive list of the popular attractions in Singapore, as well as some *useful features* like a MRT map and a list of useful numbers.
